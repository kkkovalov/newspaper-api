Welcome to Newspaper API

  • /articles/ - to return the list of all articles in the database

      -> returns all serialized articles

      -> no authentication required

      -> only GET requests allowed

  • /article/?id/ - to return a single article, by id

      - returns serialized article by id

      -> authentication required

      -> only GET requests allowed

  • /register/

      -> returns successfull message

      -> no auth required, but verifies that no auth is present (returns {"logged_in": True} message)

      -> only POST requests

  • /login

      -> returns JSON object {'message': 'user found', 'refresh': refresh_token, 'access': access_token}

      -> no auth required, but verifies that no auth is present (returns {"logged_in": True} message)
